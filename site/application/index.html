Setlist Page

Setlist Management

Creating a Setlist

  1. Open the ShowControl App
  2. Navigate to the Setlist tab
  3. Add songs using the interface
  4. Drag and drop to reorder
  5. Save your setlist

Setlist Features

  • Song navigation with Next/Previous controls
  • Progress tracking with visual feedback
  • Section markers within songs
  • Automatic cue loading

Integration with Ableton Live

The ShowControl Script communicates setlist information bidirectionally: - Send setlist from app to Live - Receive current position updates - Sync song names and sections

Display Options

Each display line can show:

Display Type Description
Scene Name Current scene/cue name
Next Scene Name Upcoming scene name
Track Name Selected track name
Looper Number Active looper index
Variation Number Current variation
Prev/Next Marker Name Marker navigation info
Setlist Song Current song in setlist
Next Setlist Song Upcoming song

User Modes

ShowControl supports up to 5 user pages, each with independent button assignments.

Button Actions

Each of the 5 buttons (+ 5 shifted) can be assigned to:

Transport Controls

Action Description
Play/Stop Toggle playback
Play Start playback
Stop Stop playback
Record Toggle recording
Metronome Toggle click track
BPM +1 / BPM -1 Tempo adjustment
Rewind / Fast Forward Timeline navigation

Arrangement Controls

Action Description
Jump 1.1.1 Jump to start
Arr. Rec Arrangement recording
Arr. Loop Toggle arrangement loop
Prev/Next Marker Navigate markers
Loop to Next Marker Set loop to next marker

Session Controls

Action Description
Session Rec Session recording
Prev/Next Scene Scene navigation
Launch Scene Trigger current scene
Prev/Next Section Section navigation
Loop Section Loop current section

Looper Controls

Action Description
Looper State Toggle looper
Prev/Next Looper Navigate loopers
Clear Clear current looper
Clear All Clear all loopers

Custom MIDI

Action Description
Custom MIDI Send custom Note/CC/PC message

Shift Function

Press the Shift button to access a secondary layer of 5 additional button assignments (buttons 6-10).

Encoder Actions

The rotary encoder can be assigned to:

Action Description
BPM Adjust tempo
Scene Select Navigate scenes
Track Select Navigate tracks
Scroll General scrolling
Selected Parameter Adjust selected parameter
Track Volume/Pan Mix controls
Track Send A/B Send levels

The encoder push button has separate assignable actions.

Pedal Configuration

Both pedal inputs support: - Momentary mode (sends value on press/release) - Toggle mode (alternates on/off) - Custom MIDI messages

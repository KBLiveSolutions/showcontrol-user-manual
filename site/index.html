Show Control

Introduction

ShowControl is an advanced MIDI control system designed for professional live performance control of Ableton Live and other DAWs. It consists of three main components:

  • ShowControl Firmware - Runs on Raspberry Pi Pico hardware controller
  • ShowControl App - Desktop application (Electron) for configuration and monitoring
  • ShowControl Script - Ableton Live Remote Script for DAW integration

Key Features

  • Bidirectional MIDI/OSC communication with DAWs
  • Customizable user pages with buttons, encoders, and pedals
  • Setlist management for live performances
  • Ethernet connectivity for reliable OSC communication
  • LED feedback with customizable colors
  • TFT display for visual feedback

Installation

Firmware Installation

Desktop App Installation

Ableton Live Script Installation

  1. Copy the __showcontrol folder to:
  2. macOS: ~/Music/Ableton/User Library/Remote Scripts/
  3. Windows: \Users\[username]\Documents\Ableton\User Library\Remote Scripts\
  4. In Ableton Live, go to Preferences > Link/Tempo/MIDI
  5. Select ShowControl as a Control Surface

Hardware Setup

Network Configuration

ShowControl supports Static IP configuration.

Configure manually via the desktop app: - IP Address: Device IP (e.g., 192.168.1.100) - Port: OSC port (default: 42050)

Configure on the controller in the Settings Menu

MIDI Setup

ShowControl appears as a USB MIDI device when connected via USB. It supports: - Note messages (0-127) - Control Change (CC) messages (0-127) - Program Change (PC) messages - SysEx messages for configuration

OSC & MIDI Communication

OSC Messages

Incoming Messages (to Ableton Live)

Address Description
/live/test Connection test
/live/api/reload Reload script code
/showcontrol/ping Connection ping

Outgoing Messages (from Ableton Live)

Address Description
/fromLive/ping Ping response
/fromLive/newSetLoaded New set loaded
/live/error Error messages

OSC Ports

Port Direction Default
Listen Incoming 42050
Response Outgoing 42051

MIDI Message Types

Type Range Usage
Note 0-127 Button triggers
CC 0-127 Continuous controls
PC 0-127 Program changes

LED Color Feedback

Buttons provide LED color feedback based on Ableton's color palette (64 colors). Colors are automatically synchronized with assigned functions.

User Mode Configuration

  1. Select User tab in the app
  2. Choose user mode (1-5) from dropdown
  3. Click on buttons to assign actions
  4. Use Shift to configure shifted layer
  5. Configure encoder turn/push actions
  6. Set display line content

Saving Configuration

  • To device: Changes are sent via MIDI SysEx
  • To file: Export/import JSON configuration
  • Persistent storage: Device saves to internal flash (LittleFS)

Configuration File Structure

Configuration is stored in data.json:

{
  "options": {
    "useDHCP": true,
    "nightMode": false,
    "selectedPage": 0,
    "userPagesAmount": 3,
    "ledBrightness": 5,
    "displayBrightness": 80,
    "port": 42050,
    "manualIP": [192, 168, 1, 100]
  },
  "pages": [
    {
      "controls": [...],
      "encoder": [...],
      "display": [...]
    }
  ]
}

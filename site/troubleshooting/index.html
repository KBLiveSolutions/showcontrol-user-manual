Troubleshooting

Connection Issues

Device Not Detected via USB

  1. Check USB cable (use data cable, not charge-only)
  2. Try different USB port
  3. Restart the device
  4. Check Device Manager (Windows) or System Information (macOS)

No OSC Communication

  1. Verify network connection (check ETH indicator on display)
  2. Confirm IP addresses match between device and app
  3. Check firewall settings allow OSC ports
  4. Test with /live/test message

MIDI Not Working

  1. Verify MIDI port selection in DAW
  2. Check ShowControl appears as MIDI device
  3. Send /live/api/reload to restart script

Display Issues

Blank Display

  1. Check TFT_LED pin connection
  2. Verify display initialization in logs
  3. Try power cycling the device

Corrupted Display

  1. May indicate SPI timing issues
  2. Check for interference with other USB devices
  3. Update firmware to latest version

Configuration Issues

Settings Not Saving

  1. Verify filesystem is properly uploaded
  2. Check LittleFS initialization in logs
  3. Re-upload filesystem: pio run --target uploadfs

JSON Errors

  1. Use jsonManager.printDiagnostics() for debugging
  2. Check for corrupted data.json
  3. Reset to default configuration

Log Monitoring

Serial Monitor

pio device monitor --baud 115200

Enable Debug Output

Debug modes are available in config/consts.h: - DEBUG_SETUP - Initialization logs - DEBUG_ETHERNET - Network logs - DEBUG_DISPLAY - Display logs

Getting Support

  1. Open an issue on GitHub
  2. Include firmware version and logs
  3. Describe steps to reproduce the problem
  4. Use /live/test to verify connectivity

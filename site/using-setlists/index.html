Using Setlists¶

Goal¶

Create and operate a deterministic Song/Section structure for live performance.

Preconditions¶

  • Device connected and stable
  • Source context available (Scenes/Markers or equivalent workflow)

1) Build setlist structure¶

  1. Open Setlist tab.
  2. Select source (Scenes or Markers).
  3. Create/select target setlist.
  4. Drag content into Songs.
  5. Organize Sections inside Songs.

Verify¶

  • Song order matches show order
  • Section boundaries are meaningful for jumps/loops

2) Refine and validate¶

  • Reorder Songs/Sections as needed.
  • Use undo/redo while editing.
  • Review durations and transition points.

Verify¶

  • no duplicate/misplaced Song or Section entries
  • navigation is repeatable in rehearsal

3) Live operation guidance¶

During show operation:

  • avoid structural edits mid-song
  • keep transition actions deterministic
  • use known-safe jump points

If setlist behavior is wrong¶

Go to Troubleshooting.

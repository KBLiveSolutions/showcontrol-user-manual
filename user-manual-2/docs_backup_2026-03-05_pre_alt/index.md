# ShowControl User Manual

Welcome to the ShowControl manual.

ShowControl is a live-performance control system built around:

- **ShowControl hardware controller**
- **ShowControl desktop app**
- **Ableton Live remote script** (optional but recommended for deep integration)

This manual focuses on how to use what is already implemented in the current app and hardware workflow.

---

## What you can do with ShowControl

- Build and organize **setlists** from Ableton Scenes or Arrangement Markers
- Create and manage songs/sections with drag & drop
- Use **undo/redo** while editing setlists
- Configure **user pages** (buttons, encoder, pedals, display)
- Adjust device **settings** (LED brightness, display brightness, IP/port)
- Monitor **USB/ETH status** directly in the app top bar
- Access built-in manual pages from the **Info** tab

---

## Application tabs (current layout)

The desktop app currently includes:

1. **Info** (built-in documentation)
2. **Setlist** (main editing workspace)
3. **Performance** (performance-oriented controls/display)
4. **User** (custom controls and mappings)
5. **Settings** (device + network settings)

---

## Recommended first-time setup

1. Connect the controller via **USB** or **Ethernet**
2. Launch the desktop app
3. Confirm connection status in the top bar (**USB** and/or **ETH** dots)
4. Open **Settings** and verify:
   - LED brightness
   - Display brightness
   - IP address / port (if using network)
5. Open **Setlist** and choose **Scenes** or **Markers** source
6. Build your first setlist by dragging entries into songs
7. Open **User** and verify button/pedal behavior for your show

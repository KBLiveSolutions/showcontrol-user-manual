# Quick Start (Performer)

This is the fastest path to a ready-to-play setup.

## Goal
Be ready to run your show with setlist navigation and mapped controls.

## 1) Connect and verify status
1. Connect controller via **USB** or **Ethernet**.
2. Open ShowControl app.
3. Check top-bar indicators:
   - **USB** should be active when using USB
   - **ETH** should be active when using network

**Expected result:** at least one transport path is clearly active.

---

## 2) Device settings (30 seconds)
1. Open **Settings** tab.
2. Set LED and display brightness to stage-friendly values.
3. If using Ethernet, confirm IP and port, then press **Send**.

**Expected result:** brightness is readable and network values are applied.

---

## 3) Setlist prep
1. Open **Setlist** tab.
2. Choose source: **Scenes** or **Markers**.
3. Create/select a setlist.
4. Drag items into songs/sections.
5. Reorder and verify structure.

**Expected result:** songs and sections reflect your show order.

---

## 4) Control validation
1. Open **User** tab.
2. Test critical mappings:
   - Play/Stop
   - Previous/Next navigation
   - Shift-layer actions
   - Encoder and pedals

**Expected result:** all critical actions behave exactly as expected.

---

## 5) Pre-show final check
- Correct setlist selected
- First song selected
- Shift state confirmed
- USB/ETH status stable
- Backup path known (see Show Day Checklist)

If any step fails, go to [Troubleshooting](troubleshooting.md).
